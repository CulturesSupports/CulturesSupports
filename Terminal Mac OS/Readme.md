To display all possible terminal commands in macOS, follow these steps:

1. **Open Terminal**: You can find the Terminal app in `/Applications/Utilities/` or use Spotlight Search by pressing `Cmd + Space` and typing "Terminal."
2. **At the Bash Prompt**: Type `Escape` key twice.
3. **Prompt to Display Commands**: You'll see a message asking, "Display all 1460 possibilities? (y or n)".
4. **Confirm**: Press the `y` key to start displaying every command available.
5. **Navigate the List**: Use the `Return` key to scroll through the list line-by-line. You can press the `Space` bar to page down an entire screen.
6. **Exit**: Press the `Delete` or `Backspace` key to return to the command prompt.

This will show you a comprehensive list of all available terminal commands on your Mac. If you want more details about a specific command, you can right-click on it and choose "Open man page" to see its manual.
