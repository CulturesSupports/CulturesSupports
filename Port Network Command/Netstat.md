You got it! `netstat` is a command-line tool that provides information about network connections, routing tables, interface statistics, masquerade connections, and multicast memberships. Here are some commonly used `netstat` commands:

1. **Display all connections and listening ports**:
   ```sh
   netstat -a
   ```

2. **Display listening ports**:
   ```sh
   netstat -l
   ```

3. **Display TCP connections**:
   ```sh
   netstat -t
   ```

4. **Display UDP connections**:
   ```sh
   netstat -u
   ```

5. **Display raw socket connections**:
   ```sh
   netstat -w
   ```

6. **Display PID and program name for each connection**:
   ```sh
   netstat -p
   ```

7. **Display routing table**:
   ```sh
   netstat -r
   ```

8. **Display network interface statistics**:
   ```sh
   netstat -i
   ```

9. **Display masquerade connections**:
   ```sh
   netstat -M
   ```

10. **Display multicast group memberships**:
    ```sh
    netstat -g
    ```

11. **Display summary statistics for each protocol**:
    ```sh
    netstat -s
    ```

12. **Display IPv6 connections**:
    ```sh
    netstat -A inet6
    ```

13. **Display extended information**:
    ```sh
    netstat -e
    ```

14. **Continuous output**:
    ```sh
    netstat -c
    ```

15. **Display network address and port numbers in numeric form**:
    ```sh
    netstat -n
    ```
